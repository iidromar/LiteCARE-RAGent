# LiteCARE-RAGent

A lightweight wearable stress detection system combining a Multi-Scale Temporal Convolutional Network with Squeeze-and-Excitation attention (LiteTCNSE), Monte Carlo Dropout uncertainty estimation, and a retrieval-augmented intervention agent.

**319,138 parameters · 1.29 MB FP32 · 8.2 ms single-thread CPU inference (Apple M4 Pro, arm64)**

---

## Architecture

```
[B, 4, 1920]  ←  60-second window @ 32 Hz (EDA, BVP, TEMP, ACC)
      │
4 × MultiScaleTCNSEBlock (dilations [1,2,4,8], channels [64,128,128,256])
  each block: CausalConv(d×1) ┐
              CausalConv(d×2) ├─ Concat → InstanceNorm → ReLU → SE → Dropout → + residual
              CausalConv(d×4) ┘
      │
GlobalAvgPool  →  [B, 256]
      │
Concat with HRV projection [B, 32]  ←  12 HRV features
      │
Linear(288, 2)  →  logits
```

MC-Dropout (N=30 passes) produces a distribution over predictions; predictive entropy drives an abstention rule: windows with entropy above threshold τ are withheld rather than misclassified.

The RAG-Mini agent retrieves context-appropriate stress interventions from a local knowledge base using tag-based scoring (no LLM required).

## Datasets

Three publicly available datasets are supported:

| Dataset | Context | Subjects | Reference |
|---------|---------|----------|-----------|
| WESAD | Lab (wrist + chest) | 15 | Schmidt et al., 2018 |
| AffectiveROAD | Driving | 13 drives | El Haouij et al., 2018 |
| EmpaticaE4Stress | Workplace | 29 | Sevil et al., 2024 |

Download each dataset and place under `data/raw/` following the structure below:

```
data/raw/
├── WESAD/            ← S2/, S3/, ..., S17/
├── AffectiveROAD_Data/
└── EmpaticaE4Stress/
```

## Installation

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Tested on Python 3.9+ with PyTorch 2.8. ONNX Runtime is optional (for `15_device_benchmark.py`).

## Quickstart

### 1 — Preprocess all datasets

```bash
python scripts/01_preprocess_wesad.py
python scripts/02_preprocess_affectiveroad.py
python scripts/03_preprocess_empatica.py
```

Each script reads from `data/raw/` and writes windowed `.npz` files to `data/processed/`.

### 2 — Train (LOSO cross-validation on WESAD)

```bash
python scripts/04_train.py --config config/config.yaml --out results/wesad_loso
```

Trains one fold per subject. Checkpoints saved as `results/wesad_loso/full_fold_<subject>.pt`.

### 3 — Full evaluation

```bash
python scripts/08_full_evaluation.py --config config/config.yaml
```

Runs all baselines (RF, BiLSTM, DilatedCNN, XGBoost, MiniRocket, InceptionTime) and the main model under identical LOSO protocol. Produces `results/full_evaluation_table.csv`.

### 4 — Cross-dataset transfer

```bash
python scripts/05_cross_dataset_eval.py   # WESAD → AffectiveROAD
python scripts/11_cross_dataset_empatica.py  # WESAD → EmpaticaE4Stress
```

### 5 — Ablation study

```bash
python scripts/06_ablation_study.py --config config/config.yaml
```

Trains and evaluates `full`, `no_se`, and `fixed_dilation` variants.

### 6 — MC-Dropout abstention evaluation

```bash
python scripts/10_abstention_eval.py
```

Calibrates per-fold τ and reports F1 before/after abstention.

### 7 — GradCAM saliency

```bash
python scripts/12_gradcam.py
```

### 8 — Device benchmark

```bash
python scripts/15_device_benchmark.py
```

Reports FP32 / INT8 / ONNX Runtime latency (500 timed iterations, 30 warmup). Must be run on the target device; ARM measurements require ARM hardware.

## Run the agent

```python
from src.utils.config import load_config
from src.models.lite_tcn_se import LiteTCNSE
from src.rag_mini.agent import LiteCARE_RAGent

cfg   = load_config("config/config.yaml")
agent = LiteCARE_RAGent("results/wesad_loso/full_fold_S2.pt", cfg, tau=0.508)

import numpy as np
window = np.random.randn(4, 1920).astype(np.float32)  # replace with real sensor data
result = agent.run(window, context="workplace")
print(result.summary())
```
